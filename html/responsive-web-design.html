What is Responsive Web Design?

We could say "responsive web design" is the solution. What, then, is the problem? The problem is that browsing the web can be done on many, many different sized devices. Even if everyone had the same computer monitor there are a variety of screen resolutions that different users may choose. Even within the group of users with the same hardware and the same screen resolution settings there are a variety of font size choices. And, of course, there are many, many choices of viewing hardware. Everything from big-screen TVs to laoptops, tablets and the latest smart phone. There are thousands and thousands of permutations of different browser window sizes. It is impractical to have a different web site design for each fo these permutations.

Responsive web design is a design philosophy whereby the size of various page elements and even the page layout itself is relative and will therefore adjust itself based on the available screen real estate. There is no need for the web site designer and developer to design or build different layouts for different screen sizes or different web pages for different devices, e.g., a dedicated "mobile" web site.

Responsive web design also means designing a web site that responds to a user's requirements as well as to a user's device requirements. So accessibility is an important feature of responsive web design (accessibility → ac1c2e3s4s5i6b7i8l9i10t11y → a11y).
